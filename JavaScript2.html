Create a web page with a button that displays a random quote each time the button is clicked. The quotes should be stored in an array, and a new quote should be chosen randomly each time the button is clicked. The web page should also display the number of times the button has been clicked. Use event handling in JavaScript to achieve this functionality. Requirements: Create an array of at least 10 quotes. Use HTML and CSS to create a button and a container for the quote and click counter. Use JavaScript to handle the button click event, and display a random quote from the array each time the button is clicked. Display the number of times the button has been clicked next to the button.
